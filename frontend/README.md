## Engines Requirement

`node: >=18.16.0`  
`npm: >=9.5.0"`

## Getting Started

```bash
# install packages
npm i

# run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:300) with your browser to see the result.
